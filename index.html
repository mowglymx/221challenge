What is 2-Minute 21-Days rule?

When you start a new habit, it should take less than two minutes to do.

The rule recognizes that simply getting started is the first and most important step toward doing something.

A new activity should feel manageable. Take for example you want to start running, then go for a 2-minute run and commit only to that. Once you start running, you will probably keep going.
